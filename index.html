Minería de Datos

La finalidad de este proyecto es definir un sitio web que le proporcione una estadística de las opiniones de una publicación de Facebook.
Esto se hace implementando inteligencia artificial que nos ayuda a analizar las publicaciones que desean ser analizadas por el usuario.
Esto sirve al usuario para poder medir una recepción de una publicación que haya hecho y brindarle una herramienta con la cual poder planear el siguiente paso de una campaña.
